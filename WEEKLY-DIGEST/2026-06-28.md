# Wochen-Digest 2026-06-28

## 🔥 Top 3 für DICH konkret

1. **Higgsfield jetzt direkt in Premiere Pro und After Effects** — Neue Adobe-Plugins sind live (Ende Mai/Juni 2026). Kein separater Browser-Tab mehr: direkt aus dem Videoeditor heraus KI-Video generieren und exportieren. Für Hofverkauf-Clips bedeutet das: Aufnahme → Schnitt → KI-Effekt in einem Fenster. Konkreter Schritt: Im Adobe-Marketplace nach „Higgsfield“ suchen und Plugin installieren.

2. **BFSG: Heute genau ein Jahr in Kraft — Bußgelder werden jetzt wirklich verschickt** — Seit 28. Juni 2025 gilt die Website-Barrierefreiheitspflicht (WCAG 2.1 Level AA). 2026 gehen die Behörden zunehmend von Verwarnungen zu echten Bußgeldern über (bis 100.000 €). Gilt ab 10 Mitarbeitenden — sobald eine Website ein Kontaktformular, Buchungsfunktion oder Download hat. Für webwerk: Jetzt ist der ideale Zeitpunkt für eine Mini-Kampagne an Bestandskunden — viele Betriebe wissen immer noch nicht, dass sie betroffen sind.

3. **Claude Code `/rewind` — kein Kontextverlust mehr nach /clear** — Neu (Juni 2026): Nach einem `/clear` kannst du mit `/rewind` die vorherige Unterhaltung komplett wiederherstellen. Bisher war nach dem Löschen alles weg. Konkreter Schritt: Einfach testen — `/clear` eingeben, dann `/rewind`.

## 🛠 Claude / KI-Tools

- **Claude Code `/rewind`** (Anthropic, Juni 2026) — Stellt die Unterhaltung vor dem letzten `/clear` wieder her. Im Alltag oft hilfreich, wenn man aus Versehen den Kontext gelöscht hat.
- **Claude Code MCP: Auto-Reconnect bei Login-Fehlern** (Anthropic, Juni 2026) — Wenn ein angeschlossenes Tool einen 401/403-Fehler zurückgibt, verbindet sich Claude Code jetzt automatisch neu. Außerdem: Beim Start wird jetzt sichtbar angezeigt, wenn ein MCP-Server noch Anmeldedaten braucht. Weniger rätselhaftes Debugging.

## 🏠 Fenster-Branche

- **BAFA-Förderung: keine Änderungen** — Weiterhin 15 % Zuschuss (bis 20 % mit iSFP-Bonus), Förderdeckel 30.000 € pro Wohneinheit/Jahr. Für Kundengespräche unverändert nutzbar.
- **BFSG-Jahrestag als Aufhänger für Kundengespräche** — Handwerksbetriebe mit Online-Auftritt + Kontaktformular + mehr als 10 Mitarbeitenden sollten ihre Website prüfen lassen. Das ist ein konkreter Gesprächseinstieg für webwerk gegenüber Fensterbetrieben in Brandenburg, die DeineFenster als Referenz kennen.
